How to use

This website allows you to extract maps from GeoFabrik data and apply aesthetic changes to create custom maps. To use the site, follow the three stages of map extraction: data, style, and serve, which can be found in the navbar above.

Step 1: Data Extraction

  • Click on the "Data" page in the navbar.
  • Select a country .pbf file from GeoFabrik.
  • Click the "mbtiles" button to download and convert the file to the correct format.
  • Wait for the download and convert time to complete.
  • You will be automatically redirected to the style page.

Step 2: Aesthetic Changes

  • Click on the "Style" page in the navbar.
  • Click on "Add layer" within Maputnik.
  • Select "map" as the data source (this is the data from the previous page).
  • Choose the desired map feature from the list.
  • Apply aesthetic changes as desired (eg. colour).

Step 3: Map Viewing

  • Click on the "Serve" page in the navbar.
  • Access Tileserver to view the completed map.
  • Note: the default viewer for Tileserver is zoomed out, so a background image is provided to help find the selected country.
  • Some geography knowledge may be required to navigate the map.
